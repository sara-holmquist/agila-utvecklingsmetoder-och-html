Klienttekniker

HTML

Html står för HyperText Markup Language och är inget programmeringsspråk, utan ett märkspråk som används för uppbyggnad och struktur av webbsidor. Det är html som anger vad som är t.ex. rubriker, listor och stycken på en webbsida. Html är uppbyggt av taggar, element och attribut som definierar vad som ska visas och hur det ska visas. Ett element består av en starttagg och nästan alltid en sluttagg, och ett innehåll mellan dessa taggar. Ett element kan ha ytligare information i sig, i form av attribut som t.ex. beskriver källan till vilken bild som ska visas eller vilken adress länken länkar till. Dessa attribut anges alltid i starttaggen.

CSS

CSS är en förkortning av Cascading Style Sheets, och är ett språk bestående av regler som beskriver hur elementen på en webbsida ska presenteras med hjälp av stilmallar. CSS kan skrivas i html dokumentet men en separat fil är att föredra för att separera strukturen/innehållet och presentationen samt att du kan använda den till flera sidor. I CSS-filen anger man t.ex. layouten, bakgrundsfärger, teckensnitt etc. Stilmallen kan även specificeras för olika mediatyper och skärmstorlekar när man vill göra en websida responsiv.

Javascript

Javascript är ett objektorienterat scriptspråk som oftast körs på klientsidan. Precis som med CSS, kan Javascript skrivas direkt i html dokumentet med <script> taggar, men av samma anledningar är en egen fil att föredra. JavaScript används till webbsidor för att t.ex. skapa funktioner och händelser samt göra sidan mer visuell.
